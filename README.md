## Цель проекта

Создание компьютерной игры в жанре метроидвания, с мрачной стилистикой.

## Используемые технологии

Для разработки игры мы используем [Unity](https://unity.com/ru) и [C#](https://learn.microsoft.com/ru-ru/dotnet/csharp/).
